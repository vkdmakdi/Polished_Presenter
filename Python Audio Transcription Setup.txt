Python Audio Transcription Setup — Quick Guide

1. Required Python Libraries
Install these via pip. All are required for the script to function correctly.

Libraries
librosa
numpy
openai-whisper
torch
soundfile

Purpose of Each
librosa — audio loading, resampling, silence detection
numpy — numerical and array operations
openai-whisper — speech-to-text engine
torch — backend for Whisper (CPU/GPU computation)
soundfile — reliable audio decoding used by librosa

Install Command
pip install librosa numpy soundfile openai-whisper torch


2. FFmpeg Installation (Windows)
Whisper and librosa rely on FFmpeg for audio processing.

Steps
Download FFmpeg from:
https://www.gyan.dev/ffmpeg/builds/
Extract the archive.
Copy the full path to the bin folder.
Example:
C:\ffmpeg\bin

3. Add FFmpeg to System PATH (Windows)
This allows FFmpeg to be accessible from anywhere.

Press Win + R
Type sysdm.cpl → Enter
Go to Advanced tab → Environment Variables
Under System variables, select Path → Edit
Click New → paste the bin path (e.g., C:\ffmpeg\bin)
Click OK → OK → OK

4. Restart Your Tools
Restart PowerShell
Restart VS Code
FFmpeg will not be recognized until terminals are restarted.

5. Verification
Run:
ffmpeg -version
If version info prints, the setup is correct.

6. Run Command (One Line)
python audio.py audio.mp3
