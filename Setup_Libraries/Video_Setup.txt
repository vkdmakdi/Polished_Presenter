Interview Analyzer — Environment & Setup Guide

1. Required Python Version
Use Python 3.9 to 3.11. Other versions are not recommended.

Check your version
python --version

2. Required Python Libraries (MANDATORY)
Install all dependencies using pip.
Install Command
pip install opencv-python mediapipe numpy tqdm

What Each Library Is Used For
opencv-python (cv2)
Video input/output, frame-by-frame processing, drawing annotations and overlays.
mediapipe
Face Mesh (eyes, mouth, eyebrows, posture landmarks) and Hands (hand movement tracking).
numpy
Numerical operations, geometry calculations, vector math.
tqdm
Progress bar for video processing.

3. Built-in Python Modules (NO installation needed)
These are included with Python by default:
argparse
json
math
time
typing

4. External Assets & System Requirements
A. Input Video Requirements
Supported formats: .mp4, .mov, .avi
Single person only
Face clearly visible
Front-facing camera preferred

B. Camera / Video Conditions (Important)
Good lighting
Minimal occlusion (no mask, no heavy face covering)
Hands visible if hand analysis is required
Stable camera (avoid handheld footage)

5. Recommended Folder Structure
interview_sim/
│
├── interview_analyzer_sim.py
├── interview.mp4
│
└── outputs/   (optional)

Auto-generated by the script
annotated_output.mp4
interview_report.json

6. OS Compatibility
Tested and supported on:
Windows 10 / 11
Linux (Ubuntu recommended)
macOS (Intel & Apple Silicon)
macOS (M1/M2) MediaPipe Fix

If MediaPipe fails to install or run:
pip install mediapipe==0.10.9

7. Run Command (One Line)
python interview_analyzer_sim.py interview.mp4
